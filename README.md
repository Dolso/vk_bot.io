Bot for vk.com
=====================
Версия vk api: 5.87
###### Данный бот поможет вам провести тест участникам вашей группы.

#### Получения токенов

Также для работы бота вам нужен **$confirmation_token** и **$token**.  
**$confirmation_token** вы можете получить зайдя на страницу вашей группы ->  
Управление -> Настройки -> Работа с API -> Callback API->  
-> Настройки сервера и вы увидите "Строка, которую должен вернуть сервер: **XXXXX** ", где **XXXXX** - $confirmation_token 

**$token** вы можете получить зайдя на страницу вашей группы -> Управление -> Настройки -> Работа с API -> Ключи доступа -> 
Cоздать ключ. Здесь вы можете выбрать все пункты, и после подтверждения личности вы получите код.

#### Работа с вопросами, вариантами ответов и ответами

В данной версии бота каждый вопрос должен содержать 4 варианта ответа. Варианты ответа мы записываем в массив **$varint**, 
который находится в начале **index.php**. Каждый эллемент данного массива - это массив, содержащий 4 варианта ответа.

Колчиство вопросов определяется по количеству эллементов массива **$protv**. В этот массив вы должны записать номера ответов по порядку.

Вопросы вы должны заполнять в массив **$vopros**.

#### Введите этот SQL код для создания таблицы в вашей, с которой будет работать бот:
CREATE TABLE IF NOT EXISTS `tik` (  
  `id` int(100) DEFAULT NULL,  
  `rank` varchar(100) DEFAULT NULL,  
  `nomer` int(11) DEFAULT NULL,  
  `pravilotv` int(100) NOT NULL,  
  `name` varchar(100) NOT NULL,  
  `popitki` int(100) NOT NULL,  
  `active` int(1) NOT NULL  
) ENGINE=MyISAM DEFAULT CHARSET=utf8;
